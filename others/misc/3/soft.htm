Es esta seccion se incluye, por primera vez, software realizado por un usuario (:
En este caso Oscar es el autor de dos frontends que, en mi opinion, son bastante buenos: su principal caracteristica es la de la facilidad de manejo! :D
El primero es un FrontEnd para el Snes9x, y el otro para el manejo mas sencillo del SpcPlay-Emu que es el que saca los sonidos de los saves del Zsnes.
Luego, para los adictos a la buena musica, aqui os exponemos la libreria de sonidos Trance de la cual la seccion Techno Music hace referencia.
Tambien os ofrecemos la version Alpha preeliminar del juego Millennium Time's que DaTaR y yo estamos realizando (:

ARCHIVO
TITULO
DESCRIPCION
s9xrun.exe
SNES9x Runer
 FrontEnd para el sencillo manejo del Snes9x, segundo mejor emulador de Snes y que supera en algunos casos al famosisimo ZSnes (:
wspcplay.exe
WinSpcPlay 1.0c
 Curioso 'FrontEnd' para manejar el SpcPlay-Emu, un reproductor de archivos SPC que son los sonidos de juegos de SNES creados por el ZSnes
milltime.exe
Millennium Time's
 Se trata del juego que estamos realizando DaTaR y yo. Aqui encontraras la version Semi-Alpha para ver como sera mas o menos la estructura de este (aunque puede variar mucho hasta su version final). Se ha de instalar forzosamente en C:\MILLTIME\
tr808.exe
Libreria Trance 8.08
 Este archivo contiene muchos samples para tus canciones de creacion propia; suplemento de Techno Music en este numero.
