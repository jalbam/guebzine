Comencemos:

Para empezar nosotros utilizaremos el Turbo C como entorno de trabajo. Las aplicaciones funcionaran bajo el S.O. DOS.

En C todas las funciones deben escribirse en minuscula, salvo algunas excepciones, que ya iremos viendo mas adelante.
Todos sus comandos estan definidos en Librerias, y para que estos funcionen, es necesario cargarlas en memoria; de lo contrario los comandos no se encontraran.
La forma de cargar en memoria una libreria es declarandola al principio de todo el programa.

Veamos este ejemplo incluido en EJEMPLO1.C: 
#include < stdio.h >
#include < conio.h >

/* Comentario interno. Programa EJEMPLO1.C */

main() /* inicia el 'cuerpo' del programa donde lo escribiremos */
{ /* Señala su lugar de apertura (MAIN) */
clrscr(); /* Limpia la pantalla: CLeaR SCReen. Similar al CLS del DOS y Basic. */
printf("Hola Mundo!!!"); /* Muestra el mensaje que esta entre comillas en pantalla */
getch(); /* Realiza una pausa hasta que se pulse una tecla */
} /* Finaliza MAIN y el programa ya que no hay mas debajo */

Fijemosnos que los comentarios internos se abren con /* y se cierran con */.
Las librerias a declarar son stdio.h y conio.h. Es de destacar su extension: .h. Los espacios que se han añadido entre los simbolos < y > que aguardan el nombre de las librerias son inexistentes, ya que en lenguaje HTML no he encontrado otra forma de que se mostrase texto en pantalla comprendido entre esos caracteres a causa de como este esta estructurado. No los pongais.
A continuacion, se abre el 'menu' MAIN, que es el standard que mas es utilizado, y sobre todo en estos programillas tan sencillos.
Luego borramos la pantalla e imprimos el texto 'Hola Mundo!!!'. Se realiza una pausa para que el usuario pueda ver el mensaje antes de que se vuelva al Turbo C.
Para compilarlo y ejecutarlo utilizamos las teclas Control + F9 .
