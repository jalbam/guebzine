En esta entrega sobre la programación de Visual Basic vamos a empezar con algo muy sencillo. Un programa que se rueda en un pantallas (nombre que se le da al Windows) consta de un formulario o ventana gráfica y una programación, pues bien, con el Visual Basic se realizan estas dos partes del programa de una forma muy sencilla, a continuación os explico una forma de programar nuestro "hola mundo" bajo Visual Basic.

 

 

1.-En la plantilla de formulario eligiremos de la barra de herramientas un objeto de texto, pulsando sobre este icono .

2.-Nos dirigiremos hacia el formulario y pinchando con el mouse arrastraremos este haciaendo un rectangulo, cuando dejemos de pulsar el ratón, nos aparecerá un cuadro de texto en nuestro formulario.

3.-Ahora de la barra de herramientas eligiremos un pulsador o botón, pulsando sobre este icono .

4.-Repetiremos el paso 2 pero en este caso nos saldrá un botón.

5.-Ahora queda ponerle el código que queremos que ejecute el botón cuando este sea pulsado, para hacer esto no situamos con el ratón sobre el y hacemos doble click, esto hace abrirse una pantalla nueva de texto, cuando agamos esto nos tendrá que aparecer un mensage más o menos como el siguiente:

Private Sub Command1_Click()

End Sub

6.-En donde está el puntero del texto (en medio de los dos textos de azul) teclearemos lo siguiente:

Text1.Text = "Hola mundo"

7.-Si ya has hecho todos los pasos entonces solo te queda irte al menú "Ejecutar" y elegir la opción "Iniciar", esto ejecutará tu primer programa, cuando pulses sobre el botón aparecerá el texto "hola mundo" (sin las comillas).
