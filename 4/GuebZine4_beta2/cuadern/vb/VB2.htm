Un objeto de Visual Basic tiene la extensión *.ocx, estos objetos se realizan con el fin de que un usuario pueda usar controles de una forma fácil y sencilla, así nos ahorraremos mucho código. Ya se que muchos de vosotros prefieren hacerse su propio código para asegurarse de su correcto funcionamiento pero usar un control que ya esta hecho no es nada malo, yo también me hago mis propios códigos, ahora por ejemplo estoy haciendo un servidor de IRC y ví muchos controles sobre el tema pero no me convencieron de todo, así que lo que hago es mi propio control o mi propio código adaptado a mis esigencias.(Control = Objeto).

Como con todos los lenguajes de programación, el Visual Basic también tiene teoría, hoy os vamos a enseñar un poco cuales son las propiedades de algunos objectos que vienen incorporados en el paquete de Visual Basic.

El Formulario.

(Name) Es el texto con el que le vas a llamar al control cuando estés escribiendo código.
Caption Sirve para introducir el nombre del programa.
BorderStyle Opciones de la 1 a la 5 para poner distintos bordes al formulario.
BackColor Sirve para poner el color de fondo.
Icon Este será el icono que tenga el programa.
Picture Con esta propiedad podremos insertar una imagen como fondo de nuestro formulario.
ShowInTaskbar Esta propiedad permite salir en la barra de tareas(valor=true) o no(valor=false).
Visible Acción que hace que el control se visualice(valor=true) en el formulario o no(alor=false).
WindowState Con esta propiedad se define la forma de como se va a situar el formulario inicialmente, abierto con la forma que lo programamos(valor=0), minimizado(valor=1), maximizado(valor=2).

Un cuadro de texto denominado Text.

(Name) Es el texto con el que le vas a llamar al control cuando estés escribiendo código.
Alignment Sirve para justificar el texto a la izquierda(valor=0), a la derecha(valor=1), o centrado(valor=2).
Locked Sirve para bloquear el texto, bloqueado(valor=true), desbloqueado(valor=false)
Multiline Sirve para convertir el texto en más de una linea, con una linea(valor=false), más de una línea(valor=true).
Rigthtoleft Sirve para que el texto se escriba de derecha a izquierda(valor=true), o de izquierda a derecha(valor=false).
Scrollbars Cuando un texto es demasiado largo aparezca una barra de desplazamiento, anuladas(valor=0), horizontal(valor=1), vertical(valor=2), ambas(valor=3).
Text Sirve para introducir texto, el texto debe de ir entre comillas.
Tooltiptext Es el texto que aparecerá cuando el ratón esté encima del control de texto.
Visible Acción que hace que el control se visualice(valor=true) en el formulario o no(alor=false).

Un botón denominado CommandButton.

(Name) Es el texto con el que le vas a llamar al control cuando estés escribiendo código.
Caption Sirve para introducir texto, el texto debe de ir entre comillas.
Rigthtoleft Sirve para que el texto se escriba de derecha a izquierda(valor=true), o de izquierda a derecha(valor=false).
Tooltiptext Es el texto que aparecerá cuando el ratón esté encima del control de texto.
Visible Acción que hace que el control se visualice(valor=true) en el formulario o no(alor=false).
Picture Sirve para insertar una imagen en el botón.
Style Importante a la hora de insertar una imagen, la opción style debe de estar con el valor 1 para poner una imagen en el botón, por defecto está en 0.
