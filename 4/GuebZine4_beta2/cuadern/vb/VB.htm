Bienvenidos a esta primera entrega sobre Visual Basic, primero os comento que Visual Basic es un leguaje de alto nivel que sirve para la programación de diversas aplicaciones, eso sí, siempre relacionadas con los S.O. de Moco$oft, y que quede bien clara una cosa, que sea un lenguaje de programación de alto nivel no quiere decir que sea para "TORPES" o para "VAGOS", sino que es un lenguaje muy completo y con muchas aportaciones.

Un código de Visual Basic consta de tres partes:

-La primera es el nombre del objeto al que te refieres que puede ser un Text (texto), Command (botón), Label (etiqueta), etc.

-La segunda es la parte que quieres cambiar del objecto y se separa de la primera mediante un punto ".", que pueden ser Text (para cambiar el texto de un Text objecto), Caption (para cambiar el texto de un Command o de un Label), etc.

-La tercera parte es el valor que recibirá esta parte que quieres cambiar del objecto y se separa del resto por un igual "=", importante si es texto este debe de ir entre comillas.

Aquí tiene un ejemplo de código de Visual Basic:

Text1.Text = "Hola mundo"

En este ejemplo vemos claramente las tres partes por las que se compone el código, la primera "Text1", la segunda "Text" y la tercera "hola mundo" que va entre comillas puesto que es un texto.
