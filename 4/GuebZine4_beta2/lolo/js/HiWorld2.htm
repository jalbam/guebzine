Jejeje, pos vamos a hacer la técnica que aplicaré durante todo el curso. Primero el código y luego su peazo esplicación:

<SCRIPT Language="JavaScript"> function HiWorld() { alert("¡¡¡Hola Mundo!!!"); } </script> <input type="button" value="Haz k salga el alert!!!" name="HiWorld" onClick="HiWorld()">

Así a simple viste puede parecer un poquito complicado, pero no os preocupeis!!! Yo os lo explico, pa' eso me pagan (anda! no me pagan! xD)
Lo primero que hacemos, como en todo JavaScript es abrir el tag de SCRIPT para que el navegador sepa que es un JavaScript, eso no tiene más complicación. La tercera y la cuarta linia, las estudiamos en la página principal, asi que aqui tampoco nos dan problemas. La verdadera chicha, que no es tanta, está en la tercera linia.

La tercera linia nos introduce las function, las procedures como se llaman en Pascal (yo es k solo he hexo eso, supongo que en c se llamaran de otra manera xD). Una function es, sencillamente una parte de un JavaScript con uno o varios comandos preparados para ser llamados desde cualquier punto del HTML, por eso se ponen dentro del head del html, para que se carguen al principio del todo. Eso nos permite poner dos comandos seguidos y que se ejecuten cuando pulsemos un boton como en este caso.
La function consta de 3 partes: El Nombre, cualquier cosa que no tenga caracteres raros ni espacios, los parentesis, donde se definiran los comandos adicionales a ejecutar (En este caso, no tenemos que ejecutar nada, asi que, lo dejamos en blanco, ya veremos que poner allí en siguientes numeros) y, por ultimo la llave. La llave tiene que abrirse justo despues de los comandos adicionales y acabar, cuando queramos acabar la function, asi de sencillo.

Luego, para llamar la function, creamos el boton con el comando input y introducimos, el comando OnClick, que hace eso, que cuando sea clickeado, llame a la function. Sencillo, me parece ;)
